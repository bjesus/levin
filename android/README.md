# Levin Android

Android version of Levin - a simple background BitTorrent archiver.

## Features (Planned)

- Automatic torrent archiving from watch folder
- Battery-aware (pauses on battery if configured)
- Cellular-aware (pauses on mobile data if configured)
- Storage limit enforcement
- Runs in background with notification

## Requirements

- Android 7.0+ (API 24+)
- Notification permission (Android 13+)

## Building

```bash
./gradlew assembleDebug
```

## Current Status

🚧 **Work in Progress** - Phase 1: Foundation setup

This is the initial Android port of the Levin desktop daemon. See the main README for more information about the project.
